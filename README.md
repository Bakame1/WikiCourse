# 🏁 Wiki Race Multijoueur

Le but est simple : relier un article Wikipédia de départ à un article d'arrivée en cliquant uniquement sur les liens, avec le moins de clics possible.

**👉 [JOUER MAINTENANT](https://bakame1.github.io/WikiCourse/)**

## 🎮 Comment jouer ?

1. Un joueur **Crée un salon** et partage le code à 4 chiffres.
2. Les autres joueurs cliquent sur **Rejoindre** et entrent le code.
3. L'hôte définit les articles de Départ et d'Arrivée.
4. Lisez, cliquez sur les liens bleus, et soyez le premier à atteindre l'objectif !
